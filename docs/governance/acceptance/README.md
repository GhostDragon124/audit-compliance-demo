# Acceptance Evidence

This directory stores governance acceptance evidence:

- Slice acceptance reports
- Gate Review reports
- Real Provider validation reports
- Deployment acceptance reports
- Performance baselines

Acceptance reports must record real evidence. Mock tests must not be presented as real Provider validation.

Do not commit real sensitive audit materials, OCR full text, full prompts, complete model outputs, secrets, internal endpoints, uploaded files, model caches, or sensitive validation artifacts.
