## 部署手册要点
- **数据不出院**：OCR（本地 PaddleOCR/Tesseract）、LLM（127.0.0.1）、上传文件（本地磁盘）、法规索引（本地 ChromaDB）
- **OCR 安装**：PaddleOCR（推荐）或 Tesseract（备选）二选一
- **LLM 配置**：Ollama / vLLM / llama.cpp 均可，OpenAI-compatible 端点
- **验证步骤**：`curl /health` → `pytest -v` → 前后端联调
- **故障排查**：覆盖 PyMuPDF、OCR、LLM、CORS、DNS 常见问题
